# Lumen LMS

LMS - это система управления обучением, разработанная для образовательного процесса.
Система обеспечивает взаимодействие между учащимися и преподавателями через веб-интерфейс и приложения Telegram, используя Google Sheets в качестве базы данных.

## Ключевые особенности

- **Маршрутизация**: Работает на базе TanStack Router с поддержкой как кодового, так и файлового подходов к маршрутизации
- **Стиль**: Разработан с использованием Tailwind CSS для создания современного стиля, ориентированного на утилиты.
- **Компоненты пользовательского интерфейса**: Улучшены с помощью компонентов пользовательского интерфейса Shadcn для согласованного дизайна
- **Выборка данных**: Поддерживает как встроенную функциональность загрузчика TanStack Router, так и интеграцию с запросами React
- **Управление состоянием**: Включает хранилище TanStack для управления состоянием приложения с поддержкой производных состояний
- **Тестирование**: Сконфигурировано с помощью Vitest для модульного и интеграционного тестирования
- **Качество кода**: Использует Biome для компоновки и форматирования
- **Переменные среды**: Типобезопасная обработка переменных среды с помощью T3Env
